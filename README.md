## 実行環境
動作確認済みの環境は以下の通りです。
バージョンは多少前後しても問題ないはずです。

### Pythonバージョン
- Python==3.7.3

以下、requirements.txt参照

### 環境構築
- Google Colabの利用(推奨)
  - [参考記事](https://learning-with-machine.hatenablog.com/entry/2019/05/31/010459)

- ローカルに環境構築
```
$ pip install -U pip
$ pip install -r requirements.txt
```
